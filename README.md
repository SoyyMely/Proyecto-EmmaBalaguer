# Proyecto Emma

Página web del Centro Educativo Emma Balaguer de Vallejos, con sistema de administración del mismo.

## Componentes

- Inicio.
- Sobre Nosotros.
- Filosofía.
- Visítanos.
- Contáctanos.
- Noticias.
- Inicio de Sesión.
- Perfíl.
- Notificaciones.
- Administrar Noticias.
- Página no Encontrada.

## Tecnologías

- HTML
- CSS
- JavaScript
- Firebase
- Bootstrap
